GOTOWCE STYLOWANE - Rowerem po Szczytnie

Te pliki NIE wymagają assets/css/main.css z HTML5 UP.
Cały wygląd jest w assets/css/rower.css.

Jak użyć:
1. Rozpakuj ZIP.
2. Skopiuj całą zawartość rozpakowanego folderu do:
   G:\Git_Main\repozytoria\kalmar2024-45.github.io
3. Zastąp istniejące pliki.
4. W PowerShell uruchom:

cd G:\Git_Main\repozytoria\kalmar2024-45.github.io
git status
git add -A
git commit -m "Wgraj stylowana wersje strony rowerowej"
git pull --rebase origin main
git push origin main
