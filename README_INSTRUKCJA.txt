GOTOWCE - Rowerem po Szczytnie

Jak użyć:
1. Rozpakuj ZIP.
2. Skopiuj CAŁĄ zawartość rozpakowanego folderu do:
   G:\Git_Main\repozytoria\kalmar2024-45.github.io
3. Zastąp istniejące pliki, jeśli Windows zapyta.
4. W PowerShell uruchom tylko:

cd G:\Git_Main\repozytoria\kalmar2024-45.github.io
git status
git add -A
git commit -m "Wgraj gotowe pliki strony rowerowej"
git pull --rebase origin main
git push origin main
